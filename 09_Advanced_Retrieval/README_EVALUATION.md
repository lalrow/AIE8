# 📊 Retriever Evaluation Guide

## 🚀 How to Run the Complete Evaluation

### Step 1: Run Notebook Cells (In Order)

Open `Advanced_Retrieval_with_LangChain_Assignment.ipynb` and run:

```
Cell 4   → OpenAI API key setup
Cell 5   → Cohere API key setup
Cell 8   → Load PDF data from data/grade3/
Cell 12  → Create Qdrant vectorstore
Cell 13  → Prepare chunked docs for evaluation
Cell 97  → Define all 6 retriever variants
Cell 98  → Generate synthetic test dataset (SDG)
Cell 99  → (Optional) Sanity check with 3 samples
Cell 100 → Full RAGAS evaluation (6 retrievers)
Cell 101 → Export RAGAS results to JSON/CSV ✅
Cell 102 → (Optional) LangSmith dataset creation
Cell 103 → (Optional) LangSmith evaluators
```

**Expected Runtime:** ~6-8 minutes total

**Output Files After Cell 101:**
- `ragas_eval_results.json` ✅
- `ragas_eval_results.csv` ✅

---

### Step 2: Generate HTML Dashboard

After the notebook evaluation completes, run:

```bash
python generate_eval_dashboard.py
```

**Output:**
- `eval_dashboard.html` 🌐

---

### Step 3: View Dashboard

Open the dashboard in your browser:

```bash
# Linux/WSL
xdg-open eval_dashboard.html

# macOS
open eval_dashboard.html

# Windows
start eval_dashboard.html

# Or just double-click eval_dashboard.html in your file manager
```

## 🔧 Troubleshooting

### "ragas_eval_results.json not found"
→ Run notebook Cell 101 first to export the results

### "429 Rate Limit Error" during evaluation
→ Wait 60 seconds and re-run from Cell 97 (delays are already added)

### Dashboard not loading
→ Make sure `eval_dashboard.html` is in the same directory as the JSON file
